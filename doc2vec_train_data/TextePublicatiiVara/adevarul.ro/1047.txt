Microsoft va achiziţiona GitHub pentru 7.5 miliarde de dolari
5 iunie 2018, 01:06
Chris Wanstrath, CEO GitHub (stânga), alături de Nadella şi Friedman (Microsoft) FOTO Microsoft
După ce au apărut diferite informaţii de-a lungul ultimilor ani, conform cărora Microsoft era în discuţii ce priveau cumpărarea GitHub, luni, gigantul american a confirmat în mod oficial achiziţia GitHub, relatează The Verge.
Se naşte încă un gigant în telecomul american. T-Mobile şi Sprint fuzi...
GitHub a fost ultima dată evaluată la 2 miliarde de dolari, în 2015, dar Microsoft va plăti 7.5 miliarde de dolari pentru companie, tranzacţie ce se va încheia până la finalul acestui an.
GitHub furnizează instrumente de codare, fiind foarte populară printre dezvoltatori. Apple, Amazon, Google şi mulţi alţi giganţi IT folosesc serviciile companiei.
Microsoft şi-a închis propriul serviciu de profil, Codeplex, în decembrie 2017, pentru că cea mai mare parte a dezvoltatorilor migraseră către GitHub.
Această tranzacţie reprezintă o reîntoarcere la origini pentru Microsoft, dar în acelaşi timp îi oferă companiei posibilitatea de a evolua, precizează Bloomberg.
Satya Nadella, directorul executiv de la Microsoft, a declarat că „Prin alianţa cu GitHub ne întărim angajamentul faţă de libertatea, deschiderea şi inovaţia dezvoltatorilor.”
Jim Goetz, partener la Sequoia Capital, companie ce a investit în GitHub, a spus că „Această tranzacţie nu este o piatră de hotar numai pentru fondatorii şi partenerii GitHub, ci şi pentru orice dezvoltator care are o idee şi o tastatură.”
