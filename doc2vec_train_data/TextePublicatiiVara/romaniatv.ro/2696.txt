Mod de preparare New York Cheesecake
Se faramiteaza biscuitii, se pun intr-un castron adaugand untul topit si zaharul si se amesteca pana se formeaza o pasta. Se unge o tava de tort detasabila si se preseaza pasta obtinuta pe fundul tavii. Se acopera cu folie alimentara si se baga la frigider. Umplutura: Se dilueaza gelatina in apa rece si se pune la foc mic pana se dizolva. Se bate branza Philadelphia cu mixerul pana devine o crema moale. Se adauga apoi gelatina amestecand continuu pana crema devine omogena apoi se dauga zaharul si sarea. Intr-un alt castron se bate frisca pana se intareste. Se adauga peste frisca smantana grasa si apoi compozitia obtinuta se toarna usor peste branza Philadelphia. Compozitia aceasta se toarna peste blatul din biscuiti si se pune la frigider peste noapte.
Fructele de padure sau capsunile sau orice alt fruct se pun intr-o craticioara cu doua linguri de zahar si se lasa pe foc pana se obtine un fel de dulceata mai moale. Totul se trece printr-o strecuratoare si se pune peste prajitura atunci cand se serveste, adaugandu-se pe farfurie si cateva fructe
